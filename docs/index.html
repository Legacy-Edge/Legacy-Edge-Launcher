Legacy-Edge-Launcher

Download

This program makes it as simple as a double click (and using UAC one time) to get back to the legacy version of Microsft Edge.

Instructions

Just start the application by double clicking it (The exe does not need an installation).
The first time you start the application you have to grant admin rights so that the application is able to change a necessary registry key and start the new-edge-installer for changes to take affect (Have a look at the readme for more information).
After a view seconds the legacy version of Microsoft Edge should start up!
From now on you can just pin the old Edge where ever you want and delete Legacy.Edge.Launcher.exe.
